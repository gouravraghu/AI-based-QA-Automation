About Action Tracker

Action Tracker is an innovative Chrome extension designed specifically for QA Engineers to streamline automation testing processes. The extension intelligently tracks all web operations performed by users, automatically generates precise XPath selectors, and leverages AI technology to create comprehensive Playwright test cases in TypeScript.

Smart Operation Tracking

Automatically captures every user interaction on web applications

XPath Generation

Generates robust and reliable XPath selectors for each tracked element

AI-Powered Code Generation

Transforms tracked operations into professional Playwright test cases

Installation Guide

1

Download Extension

Download the Action Tracker extension file (.crx or .zip) from the official source

2

Open Chrome Extensions

Navigate to chrome://extensions/ in your Chrome browser

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page

4

Load Extension

Click "Load unpacked" and select the extension folder, or drag and drop the .crx file

5

Pin Extension

Pin the Action Tracker extension to your toolbar for easy access

Extension Controls

Start Tracking

Initiates the operation tracking mode. Once activated, the extension begins monitoring all user interactions including clicks, form inputs, navigation events, and element selections. Each action is captured with precise timing and context information.

Ready to track operations

Stop Tracking

Ceases the current tracking session and finalizes the captured operation sequence. All recorded interactions are preserved and prepared for analysis. The session data includes XPath selectors, element properties, and user action metadata.

Tracking session inactive

AI Code Generation

Processes tracked operations through AI algorithms to generate comprehensive Playwright test cases in TypeScript. The system analyzes XPath patterns, optimizes selectors, and creates maintainable test code with proper assertions and error handling.

Ready to generate test code

Execute Tests

Launches the generated Playwright test cases in your configured testing environment. Executes comprehensive test scenarios including setup, teardown, and validation steps. Provides detailed execution reports and identifies any test failures or inconsistencies.

Test environment ready

Clear Session Data

Removes all tracked operations, generated code, and session history from the extension storage. This action resets the workspace to a clean state, allowing you to start fresh tracking sessions without interference from previous data.

Session data preserved
