About Test-Pilot AI

Test-Pilot AI is an innovative Chrome extension designed specifically for QA Engineers to streamline automation testing processes. The extension tracks user actions performed on applications, generates precise locators, creates comprehensive Playwright test code with both positive and negative scenarios, and provides downloadable scripts with editable input capabilities.

User Test-Pilot AI

Tracks and records all user actions performed on applications

Locator Generation

Generates robust and reliable locators for each tracked element

Playwright Code Generation

Creates comprehensive Playwright test scripts with positive and negative scenarios

Script Download & Editing

Download generated scripts and edit user inputs for customization

Installation Guide

1

Download Extension

Download the Test-Pilot AI extension file (.crx or .zip) from the official source

2

Open Chrome Extensions

Navigate to chrome://extensions/ in your Chrome browser

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page

4

Load Extension

Click "Load unpacked" and select the extension folder, or drag and drop the .crx file

5

Pin Extension

Pin the Test-Pilot AI extension to your toolbar for easy access

Extension Controls

Start Tracking

Initiates the operation tracking mode. Once activated, the extension begins monitoring all user interactions including clicks, form inputs, navigation events, and element selections. Each action is captured with precise timing and context information.

Ready to track operations

AI Code Generation & Execution

Processes tracked operations through AI algorithms to generate comprehensive Playwright test cases in TypeScript, then automatically executes them. The system analyzes XPath patterns, optimizes selectors, creates maintainable test code, and runs the generated tests with detailed execution reports.

Ready to generate and execute test code

Clear Session Data

Removes all tracked operations, generated code, and session history from the extension storage. This action resets the workspace to a clean state, allowing you to start fresh tracking sessions without interference from previous data.

Session data preserved

Edit Input Field

Opens an interactive editor to modify user inputs, test data, and locator strategies. Allows customization of test scenarios, validation rules, and element selectors before final script generation and execution.

Editor ready for modifications

Download Generated Script

Downloads the generated Playwright test script as a TypeScript file. The script includes all positive and negative test scenarios, robust locators, and comprehensive test cases ready for execution in your testing environment.

Script ready for download
